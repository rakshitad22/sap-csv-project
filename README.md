# SAP Code Unnati Django Project

## Project Overview

This project is developed as part of the SAP Code Unnati Training Program organized by Edunet Foundation.

The application allows users to:

- Register and Login securely
- Generate CSV files dynamically
- Download CSV files using entered names
- View analytics dashboard
- Search download history
- Store user activity in database
