# 📌 Features

- User Registration and Login Authentication
- Secure Dashboard Access
- CSV File Generation and Download
- Dynamic CSV File Naming
- Download History Tracking
- Search Functionality
- Responsive Bootstrap UI
- SQLite Database Integration
- GitHub Integration
- Cloud Deployment using Render

---

# 📸 Project Modules

## 1. Login Module
Users can securely log in using username and password.

## 2. Registration Module
New users can create accounts with unique username and email validation.

## 3. Dashboard Module
Displays:
- Total Downloads
- User Email
- Last Generated CSV
- Download History

## 4. CSV Generator Module
Users can:
- Enter Name
- Generate CSV File
- Download CSV instantly

Generated CSV contains:
- Username
- Entered Name
- Email
- Download Time
